# 社区行为准则

我们希望项目社区保持尊重、理性和对事不对人。

## 期望的行为

- 尊重不同背景、经验和观点的参与者。
- 用具体事实讨论技术问题，接受有建设性的反馈。
- 对安全、隐私和法律风险保持诚实。
- 给新贡献者留出学习空间。

## 不可接受的行为

- 骚扰、歧视、侮辱、威胁或人身攻击。
- 发布他人的个人信息、位置、凭据或私密通信。
- 引导他人将项目用于欺骗、作弊、追踪、绕过风控或其他不当用途。
- 反复扰乱 Issue/PR 讨论或故意隐瞒重要风险。

维护者可以编辑、删除或拒绝违反本准则的 Issue、评论和贡献，并在必要时限制参与。如需报告，请使用 GitHub 的私下沟通或仓库安全报告渠道，避免在公开讨论中扩散敏感信息。
